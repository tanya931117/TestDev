## 函数返回值
- 有返回值 

在函数中使用return [返回值]可将函数结束，并将结果返回给调用者
- 无返回值 

若函数没有返回值，则可使用不带返回值的return，此时函数会返回None。 

若不写return，则函数默认会返回None。
